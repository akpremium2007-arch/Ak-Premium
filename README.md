# AK Store - متجر الشحن والالعاب

موقع إلكتروني احترافي لمتجر الشحن والالعاب مع نظام تقييم العملاء.

## المميزات الرئيسية

✅ **اللوغو والتنقل** - شريط تنقل عصري مع اللوغو في الزاوية  
✅ **صورة إعلانية** - صورة إعلانية احترافية (1280 × 500 بكسل)  
✅ **5 أقسام** - أقسام منتظمة مع صور وتأثيرات تفاعلية  
✅ **50 منتج** - 10 منتجات في كل قسم (الشحن، الالعاب، الاكسسوارات، الاشتراكات، العروض)  
✅ **نظام الشراء** - زر "شراء الآن" يوصل العميل مباشرة إلى WhatsApp  
✅ **تقييم العملاء** - نموذج تقييم كامل مع نجوم وحفظ التقييمات  
✅ **تصميم متجاوب** - يعمل على الهاتف والتابلت واللاب توب  
✅ **خلفية مخصصة** - صورة خلفية للموقع

## تعديل المتجر

### تغيير رقم WhatsApp

افتح ملف `script.js` وابحث عن:
```javascript
const whatsappNumber = '962XXXXXXXXX'; // استبدل بالرقم الفعلي
```

استبدل `962XXXXXXXXX` برقم هاتفك (بدون علامة +):
```javascript
const whatsappNumber = '962791234567'; // مثال
```

### تعديل المنتجات

لتعديل المنتجات، افتح `script.js` وابحث عن قسم `منتجات المتجر`:

```javascript
products: {
    title: 'شحن',
    items: [
        { id: 1, name: 'كابل شحن USB-C', price: 10, image: 'images/product-charging-1.jpg' },
        // أضف أو عدل المنتجات هنا
    ]
}
```

### تعديل معلومات التواصل

افتح `index.html` وابحث عن قسم Contact:
```html
<p>📱 واتس اب: <a href="https://wa.me/962XXXXXXXXX" target="_blank">اضغط هنا</a></p>
<p>📧 البريد الإلكتروني: info@akstore.com</p>
<p>📍 العنوان: عمّان، الأردن</p>
```

## كيفية فتح الموقع

### الطريقة الأولى: فتح مباشر
اضغط مزدوج على ملف `index.html`

### الطريقة الثانية: استخدام VS Code
- افتح VS Code
- انقر على Live Server (إذا كان مثبتاً)
- أو فتح Terminal وشغل:
```bash
python -m http.server 8000
```
ثم افتح: `http://localhost:8000`

## نصائح مهمة

📌 **جودة الصور:**

- تأكد من أن الصور بالحجم المناسب
- استخدم Canva للصور الإعلانية والأقسام

📌 **الأداء:**
- ضغط الصور الكبيرة لتسريع التحميل
- استخدم صيغة WebP للصور الحديثة

📌 **الأمان:**
- تحقق من رقم WhatsApp قبل النشر
- تجنب مشاركة بيانات حساسة

## الدعم والمساعدة

إذا واجهت أي مشاكل:

1. **الصور لا تظهر:**
   - تأكد من أن الصور موجودة في مجلد `images`
   - تحقق من الأسماء تطابق تماماً (حساس لحالة الأحرف)

2. **الروابط لا تعمل:**
   - تحقق من رقم WhatsApp صحيح
   - تأكد من توفر الإنترنت

3. **الموقع بطيء:**
   - ضغط الصور أكثر
   - حذف الصور غير المستخدمة
